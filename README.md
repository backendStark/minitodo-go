# MiniTodo

Терминальное приложение для управления задачами на Go.

## Учебный проект

Создан для изучения:

- Разработки TUI-приложений с Bubble Tea
- Слоистой архитектуры и паттернов (Repository, DI)
- Тестирования в Go
- Работы с JSON и файловой системой
- Интерфейсов и dependency injection

## Структура

```
config/     # Конфигурация
models/     # Модели данных
storage/    # Хранилище и бизнес-логика
ui/         # Терминальный интерфейс
```

## Технологии

Go 1.25.2, Bubble Tea, Lip Gloss

## Запуск

```bash
make build
./bin/minitodo
```
